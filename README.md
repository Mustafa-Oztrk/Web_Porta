# Web_Porta

Web_Porta, modern web standartlarına uygun, kullanıcı dostu ve modüler bir yapıyla geliştirilmiş bir web portalı projesidir. Kurumsal tanıtımlar, haber akışları veya kişisel portföyler için temel bir altyapı sunmayı amaçlar.

---

## 🚀 Özellikler

* **Dinamik İçerik Yönetimi:** Esnek yapısı sayesinde içerikler kolayca güncellenebilir.
* **Responsive Tasarım:** Mobil, tablet ve masaüstü cihazlarla tam uyumlu (Duyarlı) arayüz.
* **Modüler Mimari:** Temiz kod yapısı ile yeni özellikler kolayca entegre edilebilir.
* **Modern UI/UX:** Kullanıcı deneyimini ön plana çıkaran estetik ve hızlı tasarım.

## 🛠 Kullanılan Teknolojiler

Bu proje geliştirilirken aşağıdaki teknolojilerden yararlanılmıştır:

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Tasarım:** Modern Web Standartları (CSS Grid/Flexbox)
* **Versiyon Kontrol:** Git & GitHub
